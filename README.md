GlusterFS Infra
===============

The glusterfs.infra role enables you to set up gluster clusters with all gluster features.

Requirements
------------

 * GlusterFS Packages
 * Ansible version 2.3 and above

Role Variables
--------------

Example Playbook
----------------

```yaml
---
- name: glusterFS infra
  hosts: localhost
  connection: local
  gather_facts: false

```
